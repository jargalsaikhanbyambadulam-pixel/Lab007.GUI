# Агуулахын систем

## Тайлбар
Энэ төсөл нь агуулахын барааг удирдах энгийн систем юм. Java хэл дээр класс ашиглан хийсэн.

## Үндсэн боломжууд
- Бараа бүртгэх
- Агуулахад нэмэх
- Жагсаалт харах
- Тайлан гаргах

## Ажиллуулах
1. Compile хийх:
javac *.java

2. Run хийх:
java Main

## Классууд
Product, Warehouse, Clerk, Receipt, Report, InventoryCheck, Main
